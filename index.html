矩阵及其运算.md

定义1 设 $F$ 是由一些数组成的一个集合, 其中包含0和1. 如果 $F$ 中任意两个数(这两个数也可以相同)的和、差、积、商(除数不能为零)仍然是 $F$ 中的数, 那么称 $F$ 为一个数域.

Q、R、C 分别表示有理数域、实数域和复数域, 而用 $F$ 表示任意数域.

定义2 由数域 $F$ 中的 $m \times n$ 个数 $a_{ij}$ ($i=1,2, \dots, m; j=1,2,\dots,n$) 排成的 $m$ 行 ( 横的 )、$n$ 列 ( 纵的 ) 的矩形数表

称为 $F$ 上的 $m$ 行 $n$ 列矩阵或 $m \times n$ 矩阵, 简称为矩阵 ( matrix ), 记为 $(a_{ij})_{m \times n}$, 其中的 $m \times n$ 个数称为矩阵的元素 ( element ), $a_{ij}$ 表示位于这个矩阵的第 $i$ 行 ( row )、第 $j$ 列 ( column ) 处的元素, $i$ 称为元素 $a_{ij}$ 的行指标 ( row index ), $j$ 称为 $a_{ij}$ 的列指标 ( column index ).

矩阵是一个矩形数表, 加上方括号或圆括号表示这些书是一个整体. 通常用大写的黑体拉丁字母 $\bm{A}$, $\bm{B}$, $\bm{C}$, $\bm{\cdots}$ 表示矩阵. 有时为了强调矩阵的行数和列数, 通常又将 $m$ 行 $n$ 列矩阵 $\bm{A}$ 记为 $\bm{A}_{m \times n}$.

实数域和复数域上的矩阵分别简称为实矩阵 ( real matrix )和复矩阵 ( complex matrix ). 如果没有特别说明, 后面涉及的矩阵都是任意数域 $F$ 上的矩阵, 且用 $\bm{A} \in F^{m \times n}$表示 $\bm{A}$ 是数域 $F$ 上的 $m \times n$ 矩阵.
